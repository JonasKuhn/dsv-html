Aplicação do CSS

Para conseguirmos ter um bom proveito do CSS e saber quais os atributos usar para em determindo momento do código, vamos ver e entender alguns termos que mais usamos no css.


background

Esse atributo pode ser usado quando queremos adicionar uma cor de fundo, uma imagem e a transparência das mesmas.

Como existem vários atributos com border's vamos fazer uma lista (para funcionar deve-se por border + um dos atributos a seguir):

  • attachment: Possibilita adicionarmos uma imagem como uma espécie de marca D'Água.
  • image: Possibilita adicionarmos uma imagem de fundo.
  • color: Possibilita adicionarmos uma cor de fundo e sua transparência.
  • position: Configurar a posição da imagem.
  • repeat: É capas de configurar a apresentação da imagem, se quer repetir ela lado-a-lado.

border

Configura a largura, estilo e cor de todas as 4 bordas.

Como existem vários atributos com border's vamos fazer uma lista (para funcionar deve-se por border + um dos atributos a seguir):

  • bottom, right, left, top: Configura a largura, estilo e cor da borda inferior.
  • top, left, right, bottom + :
    • color: Configura a cor da borda de baixo.
    • style: Configura o estilo da citada borda.
    • width: Configura a largura da citada borda.
      • EX:
      border-bottom: ....
      border-left-color: ....

  • clear: Elementos flutuantes à esquerda ou à direita de um elemento.
  • clip: Parte visível de um elemento.
  • color: Cor de primeiro plano.
  • cursor: Tipo de ponteiro do mouse.
  • display: Se o elemento é exibido e o espaço é reservado para ele.
  • filter: Tipo de filtro aplicado ao elemento.
  • float: Se o elemento flutua, geralmente utilizado pra organizar as DIV's da página.

font

Estilo, variante, peso, tamanho e altura da linha do tipo de fonte.

  • font + :
    • family: Tipo de fonte.
    • size: Tamanho da fonte.
    • style: Permite definir o estilo da fonte.
    • weight: Peso da fonte de claro a negrito.
      • EX:
      font-family: ....

  • height: Altura exibida ao elemento.
  • @import: Folha de estilo a importar.
  • left: Posição do elemento em relação a margem esquerda da página.
  • letter-spacing: Distância entre as letras.
  • line-height: Distância entre linhas de base.
  • list-style: Tipo, imagem e posição do estilo da lista.
  • list-style-image: Marcador de item de lista.
  • list-style-position: Posição do marcador de item da lista.
  • list-style-type Marcador de item de lista alternativo.
  • margin + : Tamanho de todas as 4 margens.
    • left, right, bottom, top: Tamanho da margem referida.

    overflow: Exibição de imagens que são maiores do que suas molduras.
  • padding +: Espaço em torno de um elemento em todos os lados.
    • left, right, bottom, top: Espaço a partir da margem referida.

  • text +:
    • align: Alinhamento do texto.
    • decoration: Sublinhado, sobrelinhado ou riscado.

  • indent: Recuo da primeira linha do parágrafo.
  • transform: Transformação para todas maiúsculas, minúsculas ou inicial maiúscula.
  • visibility: Se elemento é visível ou invisível.
  • width: Largura do elemento.
